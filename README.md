# pyrlmala

The python implementation of Reinforcement Learning for Adaptive MALA.

# Requirement

- Platform
  - Ubuntu 22.04.4 LTS x86_64
- Language
  - Python 3.12.7
- Package
  - uv 0.5.1

```{bash}
uv sync
```

Please note that this project has not been tested in Windows system, consult the `Stan` and `Bridgestan`.
