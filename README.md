# Requirement

- Platform
  - Ubuntu 22.04.4 LTS x86_64
- Language
  - Python 3.12.10

```{bash}
# Download Python
uv python install 3.12
uv python pin 3.12

# Download Python Packages
uv sync
```

Please note that this project has not been tested in Windows system, consult the `Stan` and `Bridgestan`.
